# Juno Documentation

Juno is the business orchestrator for the koad:io ecosystem. This directory contains operational guides and reference material.

## Contents

- [operations/](operations/) — How Juno operates day-to-day
- [team/](team/) — Team entity reference and workflow
- [trust/](trust/) — Trust bond protocol reference
- [business/](business/) — Business model, revenue, and strategy guides

## Quick Reference

- Entity deployment: `juno invoke entity <name> "<task>"`
- Spawn interactive session: `juno spawn process <name>`
- Self-commit: `juno commit self`
- Team status: See GitHub Project koad/juno
